Getting Started with Golang​

If you're seeing this page, your Golang application was successfully deployed on Porter!

How this app works​

  • When you deployed from this repo on Porter, a Github action was triggered which detected that your application runtime was Go.
  • This Github action built a Docker image for your application that was pushed to a Docker registry in your own cloud.
  • The Docker image was deployed on your cluster with the configuration settings that you specified from the Porter dashboard.
